# my-Scrapy
a Scrapy in python

### Install
安装过程会遇到一些问题，通过升级 pip 和 单独安装 incremental 可以解决
* curl https://bootstrap.pypa.io/get-pip.py | python
* pip install scrapy
* pip install --upgrade incremental
